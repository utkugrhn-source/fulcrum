# Fulcrum — Logo + Modern Web Header Identity Packet

## Direction
Modern web-header identity, not a heavy varsity crest. The mark fuses two orthopedic ideas into one compact symbol:

1. **Andry sapling** — orthopedic historical reference: supported young tree, pole, tie points.
2. **Fulcrum / pivot** — biomechanical lifting point: small wedge and pivot dot.

The result is an editorial-medical mark suitable for a header, favicon, journal page, case library, and editor seal.

## Palette
| Token | Hex | Use |
|---|---:|---|
| Navy | `#0F2540` | Primary digital surface, wordmark, monochrome mark |
| Brass | `#B89968` | Pole, fulcrum wedge, editorial rule |
| Cream | `#F4ECDB` | Paper background, mark contrast |
| Blood red | `#9E2A2B` | F accent, pivot, rope tie detail |

## Typography recommendation
- Display: **Source Serif Pro Italic 700** or **Noto Serif Display Italic 700**
- UI/navigation: **Inter Medium/Semibold**

The SVG files include fallback stacks rather than embedded fonts. Do not ship font files inside the brand package.

## Primary usage
Use `svg/fulcrum_lockup_horizontal_navy.svg` for the modern site header direction.

Recommended header composition:
- Navy rounded editorial bar
- Mark left at 48–56 px
- Wordmark beside it
- Navigation right
- Submit/CTA as cream pill

## Minimum sizes
- Full lockup: 180 px width minimum
- Mark alone: 32 px preferred, 16 px absolute minimum
- Favicon: use `svg/favicon.svg`, plus generated PNG sizes

## Clearspace
Keep at least one pivot-dot diameter around the mark. For headers, use more horizontal space so the botanical curve does not feel cramped.

## Avoid
- Medical cross, caduceus, snake, Asclepius rod
- X-ray motifs
- Tech-startup swooshes or gradients
- Cute mascot language
- Aggressive crest/shield dominance
- Rope braid texture or faux woodgrain
